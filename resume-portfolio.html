Guitar Chord Namer

Technologies used: Vanilla JS, Botstrap

  • A complete guitar chord naming app for every possible chord built from the most popular scales.
  • Users enter 3 to 6 notes with options for different tunings & chords in sharp or flat keys.
  • The chord name, notes, intervals and other useful chord information is added to the DOM.

What was the purpose of this project?

I started this project after viewing SVG Chord generator repos on GitHub and noticed that their names were wrong.

How I built this app

I had to create a lot of variables for my form elements, output fields, music note arrays, and many “empty” array variables. Add to that a lot of for loops and if conditionals.

What did I learn building this app?

It was a challenge building the majority of the functionality inside a single function that is called from the form event listener.

Is the app complete?

This app is complete for it's purpose but it can be much better. I would like to have an option for ukeleles, banjos & mandolins, as well loading the users' tuning and sharp/flat selection. Is an app ever complete?

Writing Assistant

Technologies used: Vanilla JS, Clipboard API

  • A mouse-based text input tool for composing emails and documents.
  • Users can enter text into a text area from a virtual keyboard, or by clicking user-supplied words, phrases, and emails.
  • The Clipboard API is to copy the text area for use in email and word processing applications.

Why I built this app

It is for anyone with reduced fine motor controls who have better control clicking with an assistive mouse then with using a keyboard.

How I built this app

I split a list of common words & added them to an alphabetical mega menu; added the functionality for the keyboard keys; added the textarea with clear/copy buttons; lastly, I added the proper nouns section behind an accordion button.

What did I learn building this app?

It is absolutely essential to have a clear plan outlined of what you need to accomplish, and I learned that the hard way.

Is the app complete?

No, I need to use the File API so users can upload a file of proper nouns and sample text so that they can have a custom-built page.

Why I built a WordPress theme

To understand WordPress better, improve the page speed metrics for my websites, and to have the design and features that I wanted.

I took three Theme Dev courses on Udemy and slowly started creating my template files during each lesson of the last course. I also used the WP Theme Docs along the way.

What did I learn building the theme?

WordPress overall is impressive! The template file structure and template tags enable you to give the overall design that you want.
